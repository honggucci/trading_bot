## 사용법

```python
import json

with open("config/fib_1w_anchor.json", "r") as f:
    fib = json.load(f)

# 현재가 Fib 레벨 계산
def get_fib_level(price):
    return (price - fib["anchor"]["fib_0"]) / fib["anchor"]["range"]

# 예: $95,000 → Fib 5.24
```
